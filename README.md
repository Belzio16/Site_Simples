# Site Simples - Clínica da Mulher BIT

Este é um site simples para a Clínica da Mulher BIT, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Página inicial com seção hero e especialidades
- Design responsivo
- Efeito de hover na imagem da doutora
- Botão flutuante para WhatsApp

## Como executar
Abra o arquivo `index.html` em um navegador web.

## Estrutura do projeto
- `index.html`: Página principal
- `estilo.css`: Estilos CSS
- `script.js`: JavaScript para interatividade
- `img/`: Pasta com imagens
- `README.md`: Este arquivo onde estao as especificacoes
